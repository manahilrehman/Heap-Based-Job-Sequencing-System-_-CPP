# Heap-Based Job Sequencing System

## Overview
This project solves the classic **Job Sequencing Problem** using a **Greedy Algorithm** implemented with a **Max-Heap**. The goal is to maximize total profit by scheduling jobs such that no job exceeds its deadline.

## Features
* **Priority Queue (Max-Heap):** Efficiently picks the job with the highest profit.
* **Dynamic Job Input:** Allows users to enter any number of jobs with IDs, deadlines, and profits.
* **Greedy Logic:** Schedules jobs in the latest possible available time slot to leave room for other tasks.

## Tech Stack
* **Language:** C++
* **Concepts:** Data Structures (Priority Queue, Vectors, Structs), Greedy Algorithms.

## How to Run
1. Compile the `manahil project DSA.cpp` file using any C++ compiler (like G++ or CodeBlocks).
2. Enter the number of jobs.
3. Provide the Job ID, Deadline, and Profit for each.
4. The system will output the optimal sequence and total profit.
